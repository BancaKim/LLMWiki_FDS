# Enhancing Customer Contact Efficiency with Graph Neural Networks in Credit Card Fraud Detection Workflow

[← 카테고리](index.md) · 원문: [arXiv:2504.02275](https://arxiv.org/abs/2504.02275)

- **발표처/연도**: arXiv 2025

## 문제 (Problem)
카드 사기 탐지 후 고객에게 직접 확인 연락(customer contact)을 하는 **운영 비용** 이 큽니다.
탐지 성능을 유지하면서 **불필요한 확인 연락을 줄이는** 워크플로가 필요합니다.

## 방법 (Method)
**관계형 그래프 합성곱(RGCN, Relational Graph Convolutional Network)** 을 사기 탐지 워크플로에
통합해 거래 식별 정확도·효율을 높이고, 직접 확인 연락 필요성을 줄입니다.

## 핵심 기여 (Contributions)
- RGCN을 실제 사기 탐지 **운영 워크플로** 에 통합
- 탐지 성능 유지하면서 고객 확인 연락량 절감(운영 효율)
- 탐지 모델 + 비즈니스 프로세스의 결합 관점 제시

## 결과·데이터셋 (Results)
직접 확인 연락을 줄이면서 높은 탐지 성능 유지 보고 *(구체 수치 미확인)*.
